# Autocorrect Keyboard System

## 📌 Overview
This project implements an autocorrect keyboard system that:
- Corrects misspelled words
- Predicts the next word using context

## 🚀 Features
- Spell correction using TextBlob
- Next-word prediction using N-grams (bigrams)
- Simple NLP-based implementation

## 🛠️ Technologies Used
- Python
- NLTK
- TextBlob

## ⚙️ How It Works
1. User enters a sentence
2. System autocorrects spelling
3. Last word is used to predict next word
4. Prediction is based on frequency of word pairs
